# λ◦ (Lambda-Circle)

λ◦ is a functional programming language that employs linear types with multiplicities {0, 1, ω, &} and Lafont interaction nets as its execution model, achieving automatic memory management without garbage collection through structural resource accounting, compile-time verifiable S5' concurrency safety via graph structure verification, and a trait system with global coherence enforced at link time.
